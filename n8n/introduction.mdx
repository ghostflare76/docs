# N8N 워크플로우 자동화

N8N을 활용한 자동화 워크플로우를 시작하세요.

## N8N이란?

N8N은 강력한 워크플로우 자동화 도구로, 다양한 서비스와 API를 연결하여 자동화된 워크플로우를 만들 수 있습니다.

  </Card>
  <Card title="오픈소스" icon="code-branch">

  </Card>
  <Card title="다양한 통합" icon="plug">
    수백 가지 서비스와의 통합 지원
  </Card>
  <Card title="셀프 호스팅" icon="server">
    자체 서버에 호스팅하여 데이터 보안 강화
  </Card>
</CardGroup>

## 시작하기

<Steps>
  <Step title="N8N 설치">
    Docker 또는 npm을 통해 N8N을 설치합니다.
  </Step>
  <Step title="워크플로우 생성">
    첫 번째 워크플로우를 생성하고 노드를 연결합니다.
  </Step>
  <Step title="자동화 실행">
    워크플로우를 활성화하여 자동화를 시작합니다.
  </Step>
</Steps>
